# Обнаружение аномалий в ЭКГ-сигналах (LSTM Autoencoder)

## Описание проекта
Разработка системы поиска аномалий (Unsupervised Anomaly Detection) во временных рядах на основе датасета сердечных ритмов **MIT-BIH**. 

Модель обучается только на нормальных сигналах, обучаясь их эффективной реконструкции. Аномалии фиксируются по превышению установленного порога ошибки восстановления (MAE Loss).

* **Архитектура:** LSTM Autoencoder (PyTorch)
* **Данные:** MIT-BIH Heartbeat Dataset ([Ссылка на Kaggle](https://www.kaggle.com/datasets/shayanfazeli/heartbeat))

---

## Технический стек и подходы
* **Предобработка:** Разделение данных на Train/Val (только класс Normal) и Test (сбалансированная выборка Normal/Anomaly для оценки качества).
* **Модель:** `Encoder` (2 слоя LSTM, сжатие до `embedding_dim=64`) + `Decoder` (2 слоя LSTM + Linear слой для восстановления сигнала).
* **Функция потерь:** `nn.L1Loss` (MAE).
* **Параметры обучения:** Оптимизатор `Adam` ($LR = 10^{-3}$), 30 эпох.

---

## Результаты и метрики

* **Порог отсечения (Threshold):** `7` (MAE Loss).
* **Recall на тестовой выборке:** Успешно обнаружено **2794 из 4485** аномалий (~62.3%).

### Анализ по классам:
* **Высокая разделимость:** Модель точно фиксирует патологии классов `Ventricular (V)` и `Unknown (Q)`. Их ошибка восстановления значительно выше установленного порога.
* **Низкая разделимость:** Классы `Supraventricular (S)` и `Fusion (F)` имеют распределение ошибок, схожее с нормой. Текущая базовая архитектура для них неэффективна.

---

## Направления для улучшения
Для повышения чувствительности модели к пропускаемым классам (S и Q) планируется:
1. Протестировать 1D-CNN (сверточные автоэнкодеры) для лучшего извлечения локальных признаков сигнала.
2. Использовать динамический/адаптивный порог детекции вместо фиксированного.
